CUSTOMIZATION

Learn how to customize VELOX to match your brand identity.

COLORS

Modify the color scheme in css/styles.css:

:root {
    --primary-color: #000000;
    --secondary-color: #ffffff;
    --accent-color: #ff0000;
    --dark-color: #000000;
    --light-color: #ffffff;
}

TYPOGRAPHY

Change fonts and typography settings:

:root {
    --body-font: 'Roboto', sans-serif;
    --heading-font: 'Oswald', sans-serif;
}

IMAGES

Replace images in the assets/images/ folder with your own product photos and maintain the same file names for seamless integration.

CONTENT

Update text content directly in the HTML files. Each page has clearly structured sections for easy content modification.

COMMON ISSUES

3D effects not working?

Ensure you're using a modern browser with CSS3 support and hardware acceleration enabled.

Images not loading?

Run the template on a web server instead of opening HTML files directly in the browser.

Mobile performance issues?

The template automatically optimizes animations for mobile devices. Ensure your mobile browser supports CSS3 transforms.
