Getting Started

Requirements: Node 18+, PNPM/NPM, Git.

npm install
npm run dev
# build
npm run build && npm run preview

Folder Structure

d:/Client/landing pages/Admin Portal/
  src/
    components/ ...
    pages/ ...
    data/ ...
    lib/ ...
    hooks/ ...
  documentation/ ...
  TODO.md

Environment Variables

Copy env.example to .env and fill values.

VITE_API_URL=http://localhost:4000
VITE_DEMO_MSW=true

Scripts

npm run dev    # start dev server
npm run build  # production build
npm run preview

Theming & Customization

Customize CSS variables in tailwind.config.ts and components. Theme toggle supports system, light, dark.

Components Reference

Shadcn UI components are in src/components/ui. Higher-level components like DashboardLayout and StatsCard live in src/components.

Pages

Routes are defined in src/App.tsx. Each page is self-contained in src/pages.

Data Layer & Mock API

React Query is configured. The template will include an API client and MSW handlers for demo data.

Deployment

Build with npm run build and serve the dist/ folder via any static host.
